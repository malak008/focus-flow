# FocusFlow  🍅📚
## _This is a Pomodoro study app that helps with study sessions._ 🍅📚✨
### Features: 
- Tasks inputting: Write down your goals & tasks for the session 📝
- pixelated art theme 🌼
- 25 minutes of work, 5-minute break ⏳
- encouragement messages throughout the session 🌟

### Try it out! 🚀
1. Go to **[Releases](https://github.com/malak008/focus-flow/releases/tag/v1.0.0)** section
2. Download the ZIP file
3. Extract it into a folder
4. Open FocusFlow.exe inside the extracted folder
_Ignore any Windows security warnings_

### Inspiration 🌟
I had built FocusFlow to help students, like me, stay productive and committed to their study sessions. I made it portable & included task + time management techniques all in a clean interface.

### Technologies used ⚙
 - Electron, a desktop app framework
 - React, user interface
 - JavaScript / TypeScript
 - HTML / CSS!
